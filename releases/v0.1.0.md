Initial release of `xk6-subcommand-explore`, a k6 extension for discovering available k6 extensions for Automatic Extension Resolution.

## Overview

This extension adds an `explore` subcommand to k6 that helps discover and filter extensions from the k6 extension registry.

**Key capabilities:**
- List all available k6 extensions
- Filter by type: JavaScript, Output, Subcommand
- Filter by tier: Official, Community  
- View results as formatted tables or JSON
- View latest version and all available versions

## Usage

**List all extensions:**
```bash
k6 x explore
```

**Show compact view:**
```bash
k6 x explore --brief
```

**Output as JSON:**
```bash
k6 x explore --json
```

**Filter by type:**
```bash
k6 x explore --type javascript
```

**Filter by tier:**
```bash
k6 x explore --tier official
```

**Combine filters:**
```bash
k6 x explore --tier official --type javascript
```

**Process with jq:**
```bash
k6 x explore --json | jq '.[] | select(.tier == "official")'
```

## Future

This extension is designed to work with k6's upcoming Automatic Extension Resolution feature for subcommands, eliminating the need for manual builds.
